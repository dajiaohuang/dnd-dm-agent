# 术语表

本文件用于统一 `zh-CN/` 目录下的翻译风格。除非有明确理由，后续文件均以本表为准。

## 通用规则

- 保留 Markdown 结构、链接、表格、列表格式。
- 保留骰子表达式与规则数值，如 `1d8`、`DC 15`、`+2`。
- 法术名暂时默认保留英文原名，避免与不同中文社群译名冲突。
- 属性缩写 `STR`、`DEX`、`CON`、`INT`、`WIS`、`CHA` 若直接出现则保留；正文完整属性名翻译为中文。
