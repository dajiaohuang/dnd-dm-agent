### 操控天气

*8 环 变化系*

**施法时间：** 10 分钟

**射程：** 自身（5 英里半径）

**成分：** V、S、M（燃烧的熏香，以及混入水中的泥土和木屑）

**持续时间：** 专注，至多 8 小时

持续时间内，你控制自己 5 英里范围内的天气。你必须在户外才能施放此法术。若你移动到无法清楚通向天空的位置，法术会提前结束。

施放此法术时，你改变当前天气状况；当前天气由 GM 根据气候和季节决定。你可以改变降水、温度和风。新天气状况需要 `1d4 × 10` 分钟才会生效。一旦生效，你可以再次改变天气状况。法术结束时，天气会逐渐恢复正常。

当你改变天气状况时，在下列表格中找到当前状况，并将其阶段向上或向下改变一级。改变风时，你也可以改变风向。

**表格 - 操控天气（降水）**

| 阶段 | 状况                                      |
|-------|--------------------------------------------|
| 1     | 晴朗                                      |
| 2     | 薄云                                      |
| 3     | 阴天或地雾                                |
| 4     | 雨、冰雹或雪                              |
| 5     | 暴雨、强冰雹或暴风雪                      |
|       |                                            |

**表格 - 操控天气（温度）**

| 阶段 | 状况            |
|-------|-----------------|
| 1     | 难以忍受的酷热 |
| 2     | 炎热            |
| 3     | 温暖            |
| 4     | 凉爽            |
| 5     | 寒冷            |
| 6     | 极寒            |
|       |                 |

**表格 - 操控天气（风）**

| 阶段 | 状况          |
|-------|---------------|
| 1     | 无风          |
| 2     | 中等风        |
| 3     | 强风          |
| 4     | 大风          |
| 5     | 风暴          |
|       |               |
