### 探知术

*5 环预言系*

**施法时间：** 10 分钟

**射程：** 自身

**成分：** V、S、M（价值至少 1,000 gp 的法器，例如水晶球、银镜，或装满圣水的水盆）

**持续时间：** 专注，至多 10 分钟

你可以看见并听见一个由你选择、与你位于同一存在位面的特定生物。目标必须进行一次感知豁免，该豁免会根据你对目标的了解程度以及你与目标之间的物理联系类型进行调整。如果目标知道你正在施放本法术，且它愿意被观察，它可以自愿豁免失败。

**表：探知豁免调整值**

| 了解程度                                          | 豁免调整值 |
|---------------------------------------------------|------------|
| 间接了解（你听说过目标）                          | +5         |
| 直接了解（你见过目标）                            | +0         |
| 熟悉（你很了解目标）                              | -5         |
| 联系                                              | 豁免调整值 |
| 肖像或图片                                        | -2         |
| 所有物或衣物                                      | -4         |
| 身体部位、一缕头发、一点指甲或类似物              | -10        |
|                                                   |            |

豁免成功时，目标不受影响，且你在 24 小时内不能再次对其使用本法术。

豁免失败时，本法术会在目标 10 尺内创造一个不可见感官。你可以透过该感官看见并听见，如同你就在该处。感官会随目标移动，并在持续时间内保持在目标 10 尺内。能看见不可见物体的生物会看见该感官是一颗约你拳头大小的发光球体。

你也可以不以生物为目标，而选择一个你曾见过的地点作为本法术目标。这样做时，感官会出现在该地点且不会移动。
