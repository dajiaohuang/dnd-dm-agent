### 传送术

*7 环咒法系*

**施法时间：** 1 动作

**射程：** 10 尺

**成分：** V

**持续时间：** 立即

本法术会立即将你和至多八个射程内你能看见且由你选择的自愿生物，或一个射程内你能看见的单一物件，传送到你选择的目的地。如果你以物件为目标，它必须能够完全放入一个 10 尺立方体内，且不能由不自愿生物持握或携带。

你选择的目的地必须为你所知，且必须与你位于同一存在位面。你对目的地的熟悉程度决定你是否能成功抵达。GM 掷 d100 并查阅下表。

**表：传送熟悉度**

| 熟悉度           | 意外事故 | 相似区域 | 偏离目标 | 命中目标 |
|------------------|----------|----------|----------|----------|
| 永久法阵         | -        | -        | -        | 01-100   |
| 关联物件         | -        | -        | -        | 01-100   |
| 非常熟悉         | 01-05    | 06-13    | 14-24    | 25-100   |
| 偶然见过         | 01-33    | 34-43    | 44-53    | 54-100   |
| 只看过一次       | 01-43    | 44-53    | 54-73    | 74-100   |
| 描述             | 01-43    | 44-53    | 54-73    | 74-100   |
| 虚假目的地       | 01-50    | 51-100   | -        | -        |
|                  |          |          |          |          |

***熟悉度。*** “永久法阵”指你知道其符记序列的永久传送法阵。“关联物件”指你拥有一个在过去六个月内从目标目的地取来的物件，例如来自法师图书馆的一本书、王室套房的床单，或巫妖秘密陵墓中的一块大理石。

“非常熟悉”是你经常去过、仔细研究过，或施放本法术时能看见的地点。“偶然见过”是你见过不止一次但并不十分熟悉的地点。“只看过一次”是你看过一次的地点，可能是借助魔法看见。“描述”是你通过他人描述知道其位置和外观的地点，也许来自地图。

“虚假目的地”是不存在的地点。也许你试图探知敌人的圣所，却看见了一个幻象；或者你正尝试传送到一个熟悉但已经不复存在的地点。

***命中目标。*** 你和你的队伍（或目标物件）出现在你想去的位置。

***偏离目标。*** 你和你的队伍（或目标物件）出现在距目的地随机距离、随机方向的位置。偏离距离为原本要旅行距离的 1d10 × 1d10%。例如，如果你试图旅行 120 英里，结果偏离目标，并在两个 d10 上掷出 5 和 3，那么你偏离 15%，即 18 英里。GM 通过掷 d8 随机决定偏离方向，指定 1 为北、2 为东北、3 为东，依此按罗盘方位类推。如果你传送到一座海滨城市，却最终落在海上 18 英里处，你可能会有麻烦。

***相似区域。*** 你和你的队伍（或目标物件）最终出现在一个视觉或主题上与目标区域相似的不同区域。例如，如果你要前往自己的实验室，可能会出现在另一位法师的实验室，或一家拥有许多与你实验室相同工具和器具的炼金补给店。通常，你会出现在最近的相似地点；但由于本法术没有距离限制，理论上你可能出现在该位面任何地方。

***意外事故。*** 本法术不可预测的魔法导致一次艰难旅程。每个传送中的生物（或目标物件）受到 3d10 点力场伤害，随后 GM 在表上重新掷骰以决定你最终到达何处（可能发生多次意外事故，每次都会造成伤害）。
