### 转生术

*5 环变化系*

**施法时间：** 1 小时

**射程：** 接触

**成分：** V、S、M（价值至少 1,000 gp 的稀有油和软膏，法术会消耗该材料）

**持续时间：** 立即

你接触一个死亡人型生物，或死亡人型生物的一部分。只要该生物死亡不超过 10 天，本法术会为它形成一具新的成年身体，然后召唤其灵魂进入该身体。

如果目标的灵魂没有自由或不愿这样做，本法术失败。

魔法会为该生物塑造一具新的身体供其居住，这很可能导致该生物的种族改变。GM 掷一个 d100 并查阅下表，以决定该生物复活时采取的形态；或由 GM 选择一种形态。

**表：转生种族**

| d100  | 种族                |
|-------|---------------------|
| 01-04 | 龙裔                |
| 05-13 | 矮人，丘陵          |
| 14-21 | 矮人，山地          |
| 22-25 | 精灵，黑暗          |
| 26-34 | 精灵，高等          |
| 35-42 | 精灵，木精灵        |
| 43-46 | 侏儒，森林          |
| 47-52 | 侏儒，岩地          |
| 53-56 | 半精灵              |
| 57-60 | 半兽人              |
| 61-68 | 半身人，轻足        |
| 69-76 | 半身人，壮心        |
| 77-96 | 人类                |
| 97-00 | 提夫林              |
|       |                     |

转生后的生物记得其前世生活和经历。它保留原本形态中拥有的能力，但将原本种族替换为新种族，并相应改变其种族特质。
