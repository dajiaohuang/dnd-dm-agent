# 物件

当角色需要锯断绳索、打碎窗户或砸毁吸血鬼的棺材时，唯一明确不变的规则就是：只要有足够时间和正确工具，角色就能摧毁任何可摧毁的物件。在判定角色能否对某个物件造成破坏时，请运用常识。例如，战士能用剑砍穿一段石墙吗？不能，剑多半会先断。

就本规则而言，物件是一个独立、无生命的项目，例如窗户、门、剑、书、桌子、椅子或石块，而不是由许多其他物件组成的建筑物或载具。

## 物件的数据

当时间成为关键因素时，你可以为一个可摧毁物件设定护甲等级和生命值。你也可以赋予它对特定伤害类型的免疫、抗性和易伤。

***护甲等级***。物件的护甲等级衡量的是在攻击命中它时，对其造成伤害有多困难（因为物件没有机会闪避攻击）。下方“物件护甲等级”表为各种材质提供了建议 AC 数值。

**表格 - 物件护甲等级**

| 材质 | AC |
|---------------------|----|
| 布料、纸张、绳索    | 11 |
| 水晶、玻璃、冰      | 13 |
| 木头、骨头          | 15 |
| 石头                | 17 |
| 铁、钢              | 19 |
| 秘银                | 21 |
| 精金                | 23 |
|                     |    |

***生命值***。物件的生命值衡量的是它在失去结构完整性之前能承受多少伤害。坚固的物件通常比脆弱的物件拥有更多生命值。大型物件也往往比小型物件生命值更高，除非破坏物件的一小部分就和破坏整个物件同样有效。下方“物件生命值”表为大型及以下的脆弱物件和坚固物件提供了建议生命值。

**表格 - 物件生命值**

| 体型 | 易碎 | 坚韧 |
|-------------------------------------|----------|-----------|
| 微型（瓶子、锁）                    | 2（1d4）  | 5（2d4）   |
| 小型（箱子、鲁特琴）               | 3（1d6）  | 10（3d6）  |
| 中型（桶、枝形吊灯）              | 4（1d8）  | 18（4d8）  |
| 大型（推车、10 尺乘 10 尺的窗户）  | 5（1d10） | 27（5d10） |
|                                     |          |           |

***巨型与超巨型物件***。普通武器对许多巨型和超巨型物件几乎没什么用，例如巨大的雕像、高耸的石柱或庞然巨石。话虽如此，一支火把也能烧毁一幅巨型挂毯，而一个 *earthquake* 法术足以把巨像震成瓦砾。若你愿意，可以追踪巨型或超巨型物件的生命值；你也可以直接裁定该物件能抵挡当前武器或力量多久。如果你要追踪其生命值，请将它分割成大型或更小的区域，并分别记录各区域生命值。摧毁其中一个区域，可能就足以毁掉整个物件。例如，一尊超巨型人类雕像在它的一条大型腿部生命值降为 0 时，就可能会倒塌。

***物件与伤害类型***。物件免疫毒素伤害和心灵伤害。你也可以裁定某些伤害类型对特定物件或材质比其他类型更有效。例如，钝击伤害很适合砸碎东西，却不适合切断绳索或皮革。纸或布制品可能对火焰伤害和闪电伤害易伤。鹤嘴锄能凿开石头，却无法有效砍倒树木。和往常一样，请运用你的最佳判断。

***伤害阈值***。诸如城墙这类大型物件通常具有额外的韧性，以伤害阈值表示。拥有伤害阈值的物件会免疫所有伤害，除非单次攻击或效果造成的伤害数值等于或高于该伤害阈值，此时它才会正常承受伤害。任何未达到或超过物件伤害阈值的伤害都视为表面损伤，不会减少其生命值。
