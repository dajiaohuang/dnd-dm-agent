# 疯狂

在一场典型战役中，角色通常不会因为自己日复一日面对的恐怖与制造的杀戮而发疯，但有时，作为冒险者所承受的压力确实会超出一个人所能承受的极限。如果你的战役带有强烈的恐怖主题，你可以使用疯狂规则来强化这种氛围，突出冒险者所面对威胁的极端可怖。

## 陷入疯狂

各种魔法效果都可能让原本稳定的心智陷入疯狂。某些法术，例如 *contact other plane* 和 *symbol*，都可能导致精神错乱；你可以使用此处的疯狂规则来替代这些法术原本的效果。疾病、毒药，以及诸如灵能风暴或癫狂魔域呼啸狂风之类的位面效应，也都可能造成疯狂。一些神器同样会摧毁使用它们或与它们调谐者的精神。

抵抗导致疯狂的效果，通常需要进行一次感知或魅力豁免。

## 疯狂效果

疯狂可以是短期、长期或永久性的。大多数相对普通的效果会施加短期疯狂，持续仅数分钟。更为恐怖的效果，或多种效果叠加时，则可能导致长期疯狂或永久疯狂。

受到 **短期疯狂** 的角色，会承受“短期疯狂”表中的某项效果，持续 1d10 分钟。

受到 **长期疯狂** 的角色，会承受“长期疯狂”表中的某项效果，持续 1d10 × 10 小时。

受到 **永久疯狂** 的角色，会从“永久疯狂缺陷”表中获得一个新的角色缺陷，直到被治愈为止。

**表格 - 短期疯狂效果**

| d100 | 效果（持续 1d10 分钟）                                                                                                  |
|--------|------------------------------------------------------------------------------------------------------------------------------|
| 01-20  | 角色退缩进自己的内心并陷入麻痹。若角色受到任何伤害，该效果结束。                                                             |
| 21-30  | 角色陷入失能，并在持续时间内尖叫、大笑或哭泣。                                                                               |
| 31-40  | 角色陷入恐慌，并且每回合都必须使用其动作和移动远离恐惧来源。                                                                 |
| 41-50  | 角色开始胡言乱语，无法正常说话或施法。                                                                                       |
| 51-60  | 角色每回合都必须使用其动作攻击最近的生物。                                                                                   |
| 61-70  | 角色产生逼真的幻觉，属性检定具有劣势。                                                                                       |
| 71-75  | 角色会去做任何人告诉他或她去做、且明显不会自我毁灭的事。                                                                     |
| 76-80  | 角色会产生难以抗拒的冲动，想吃下泥土、黏液或腐肉等怪异之物。                                                                 |
| 81-90  | 角色陷入震慑。                                                                                                               |
| 91-100 | 角色失去意识。                                                                                                               |

| 91-95  | 角色失去说话能力。                                                                                                                                                                      |
| 96-100 | 角色失去意识。无论如何摇晃或对其造成伤害都无法使其醒来。                                                                                                                                |

### 治愈疯狂

*calm emotions* 法术可以压制疯狂的影响，而 *lesser restoration* 法术可以移除角色的短期疯狂或长期疯狂。根据疯狂的来源，*remove curse* 或 *dispel evil* 也可能有效。要移除永久疯狂，则需要 *greater restoration* 或更强大的魔法。
