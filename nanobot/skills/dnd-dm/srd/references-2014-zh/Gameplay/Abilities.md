# 属性

六项属性可以快速描述每个生物的身体与精神特征：

- **力量**，衡量身体力量
- **敏捷**，衡量灵活性
- **体质**，衡量耐力
- **智力**，衡量推理与记忆
- **感知**，衡量察觉与洞悉
- **魅力**，衡量人格力量

一个角色是肌肉发达且洞察敏锐？还是才智超群又魅力十足？抑或身手灵活且体格强健？属性值定义了这些特质，也定义了一个生物的长处与弱点。

游戏中的三种主要掷骰，属性检定、豁免和攻击检定，都依赖这六项属性值。本书导言已说明这些掷骰背后的基本规则：掷一个 d20，加上由六项属性之一导出的属性调整值，再将总数与目标数值比较。

## 属性值与调整值

每个生物的属性都有一个数值，用以定义该能力的强弱。属性值不仅衡量天生能力，也涵盖与该属性相关活动中的训练与熟练程度。

10 或 11 是普通人类的平均水平，但冒险者与许多怪物在大多数属性上都高于平均。18 通常是一个人所能达到的最高水平。冒险者的属性值可高达 20，而怪物与神性存在则可高达 30。

每项属性还对应一个调整值，由属性值推导而来，范围从 -5（属性值为 1）到 +10（属性值为 30）。“属性值与调整值”表列出了 1 到 30 属性值所对应的调整值。

| 6-7   | -2       |

| 12-13 | +1       |

由于属性调整值几乎会影响每一次攻击检定、属性检定和豁免，因此在实际游戏中，属性调整值出现得比对应属性值更频繁。

# 优势与劣势

有时某项特殊能力或法术会告诉你：你在属性检定、豁免或攻击检定上具有优势或劣势。当这种情况发生时，在进行该掷骰时额外再掷一个 d20。若你具有优势，则采用两个结果中较高的那个；若你具有劣势，则采用较低的那个。例如，若你具有劣势并掷出 17 和 5，则你使用 5。若你具有优势并掷出这两个数字，则你使用 17。

若有多个情境影响一次掷骰，而其中一些给予优势、一些施加劣势，你依然不会额外掷超过一个 d20。举例来说，即使有两个有利情境给予优势，你也只额外再掷一个 d20。

如果情势同时导致一次掷骰既有优势又有劣势，则视为两者都没有，你只掷一个 d20。即使有多个情境带来劣势，而只有一个给予优势，反之亦然，规则也相同。在这种情况下，你既不具有优势，也不具有劣势。

当你具有优势或劣势，而游戏中的某项效果，例如半身人的幸运特质，又允许你重掷或替换 d20 时，你只能重掷或替换其中一个骰子，由你选择。例如，若一个半身人在属性检定上具有优势或劣势，并掷出 1 和 13，则该半身人可以使用幸运特质重掷那枚 1。

通常，你会通过特殊能力、动作或法术获得优势或劣势。激励也能给予角色优势。GM 也可以裁定环境会将掷骰朝某个方向推移，并因此给予优势或施加劣势。

# 熟练加值

角色拥有由等级决定的熟练加值。怪物也拥有该加值，它会被直接整合进它们的数据栏。熟练加值会在属性检定、豁免和攻击检定的规则中使用。

你的熟练加值不能在同一次掷骰或同一个数值上被加入超过一次。例如，如果两条不同规则都说你可以将熟练加值加入一次感知豁免，那么你在进行该豁免时仍然只加入一次熟练加值。

有时，在应用熟练加值之前，它可能会被乘除（例如翻倍或减半）。例如，游荡者的“专精”特性会令某些属性检定的熟练加值翻倍。若某种情形让你的熟练加值看起来会在同一次掷骰中应用多次，你依然只加入一次，并且也只进行一次乘除。

同样地，如果某项特性或效果允许你在进行某次属性检定时乘算熟练加值，而该检定本来并不会获得熟练加值，那你依然不能把熟练加值加入该检定。因为在这次检定中，你的熟练加值视为 0，而 0 无论乘以多少仍是 0。举例来说，若你不熟练历史技能，那么即使某项特性允许你在进行智力（历史）检定时让熟练加值翻倍，你也不会从中获得任何收益。

一般而言，你不会将熟练加值乘用于攻击检定或豁免。若某项特性或效果允许你这样做，则适用同样规则。

# 属性检定

属性检定用来测试角色或怪物与生俱来的天赋和训练，以尝试克服某项挑战。当角色或怪物尝试一项有失败可能性的行动（非攻击）时，GM 会要求进行属性检定。若结果并不确定，就由骰子来决定。

对于每一次属性检定，GM 都要决定六项属性中哪一项与当前任务最相关，并设定该任务的难度，以难度等级（DC）表示。

任务越困难，其 DC 就越高。“典型难度等级”表列出了最常见的 DC。

**表格 - 难度等级**

| 任务难度 | DC |
|-------------------|----|
| 非常容易         | 5  |
| 容易              | 10 |
| 中等            | 15 |
| 困难              | 20 |
| 非常困难         | 25 |
| 几乎不可能 | 30 |
|                   |    |

进行属性检定时，掷一个 d20 并加入相关属性调整值。与其他 d20 掷骰一样，应用加值和减值后，将总数与 DC 比较。若总数等于或超过 DC，则属性检定成功，生物克服了当前挑战。否则即为失败，这意味着角色或怪物没有朝目标取得进展，或者只取得了伴随代价的进展，具体由 GM 裁定。

## 对抗检定

有时，一个角色或怪物的努力会与另一个角色或怪物的努力直接对抗。这会发生在双方都试图做同一件事且只有一方能成功时，例如争抢一枚掉在地上的魔法戒指。这种情况也适用于一方试图阻止另一方达成目标，例如怪物想撞开一扇门，而冒险者正顶着门不让它打开。在这类情境中，结果由一种特殊形式的属性检定决定，称为对抗检定。

对抗中的双方都进行符合各自努力的属性检定。他们会应用所有合适的加值与减值，但不再与 DC 比较，而是彼此比较总数。总数较高的一方赢得对抗。该角色或怪物要么成功完成该行动，要么阻止另一方成功。

若对抗结果平手，则局势维持对抗发生前的状态。因此，其中一方可能会因现状而“默认获胜”。例如，两名角色平手争抢地上的戒指，则谁也没拿到。在怪物试图撞开门与冒险者试图顶住房门的对抗中，平手意味着门仍然关着。

## 技能

每项属性都涵盖广泛能力，其中包括角色或怪物可能熟练掌握的技能。技能代表某项属性的一个具体侧面，而一个个体对某项技能的熟练，说明他在该侧面上有专长。（角色的初始技能熟练在角色创建时决定，怪物的技能熟练则会出现在怪物数据栏中。）

例如，一次敏捷检定可能体现角色做出杂技动作、顺手藏物，或保持隐藏的尝试。敏捷的这些侧面分别对应三个技能：杂技、巧手与隐匿。因此，若一个角色熟练隐匿技能，那么他在与潜行和躲藏相关的敏捷检定上会表现得特别出色。

各属性对应的技能如下所示。（体质不对应任何技能。）在本节后面的各属性说明中，可见该属性相关技能的使用示例。

**表格 - 相关技能**

| 力量 | 敏捷 | 智力 | 感知 | 魅力 |
|-----------|-----------------|---------------|-----------------|--------------|
| 运动 | 杂技 | 奥秘 | 驯兽 | 欺瞒 |
| | 巧手 | 历史 | 洞悉 | 威吓 |
| | 隐匿 | 调查 | 医药 | 表演 |
| | | 自然 | 察觉 | 游说 |
| | | 宗教 | 求生 | |

有时，GM 会直接要求你进行某种技能对应的属性检定，例如“做一次感知（察觉）检定”。另一些时候，玩家可能会询问 GM，某项技能熟练是否适用于当前检定。无论哪种情况，只要某项技能熟练适用，该个体就可以将熟练加值加入涉及该技能的属性检定中。若不熟练，则进行一次普通属性检定。

例如，若一个角色尝试攀登一处危险山崖，GM 可能会要求进行一次力量（运动）检定。若该角色熟练运动技能，则其熟练加值会加入该力量检定；若不熟练，则他只进行一次力量检定。

### 变体：不同属性搭配技能

通常，一项技能熟练只适用于某一种属性检定。例如，运动通常适用于力量检定。但在某些情况下，你的技能熟练也可能合理地适用于另一种检定。在这些情形中，GM 可能会要求使用一种不寻常的“属性 + 技能”组合，或者你也可以询问 GM，自己是否能将某项熟练应用到另一种检定中。举例来说，若你必须从离岸小岛游回大陆，GM 可能会要求你进行一次体质检定，看你是否有足够耐力游那么远。在这种情况下，GM 可能允许你应用运动熟练，并要求进行一次体质（运动）检定。因此，如果你熟练运动，便可以像平常进行力量（运动）检定那样，把熟练加值加入这次体质检定。同样地，当你的半兽人野蛮人用纯粹力量来恫吓敌人时，GM 可能会要求进行一次力量（威吓）检定，即便威吓通常与魅力相关。

## 被动检定

被动检定是一种特殊的属性检定，不涉及任何掷骰。它可以代表反复进行同一任务时的平均结果，例如一遍又一遍搜索暗门；也可用于 GM 想秘密判定角色是否成功，而不公开掷骰的情形，例如察觉一个隐藏的怪物。

角色的被动检定总值计算如下：

`10 + 平常适用于该检定的所有调整值`

若该角色在该检定上具有优势，则加 5；具有劣势，则减 5。游戏将一个被动检定总值称为 **分数**。

例如，若一个 1 级角色感知为 15，且熟练察觉，那么他的被动感知（察觉）分数为 14。

“敏捷”部分中有关隐藏的规则，以及探索规则，都依赖被动检定。

## 协作

有时，两个或更多角色会联手完成一项任务。负责主导努力的角色，或者属性调整值最高的那个角色，可以带着优势进行一次属性检定，以反映其他角色提供的帮助。在战斗中，这需要使用帮助动作。

一个角色只有在该任务本来就是他能够单独尝试的前提下，才能提供帮助。例如，撬锁需要熟练使用盗贼工具，因此不具备该熟练的角色便无法在这件事上帮助别人。此外，只有当两人或更多人共同合作确实会提高效率时，角色才能提供帮助。某些任务，比如穿针引线，有人帮忙也不会变得更容易。

### 群体检定

当一群人正共同尝试达成某事时，GM 可能会要求进行一次群体属性检定。在这种情况下，擅长该任务的角色会帮助那些不擅长的人。

进行群体属性检定时，队伍中的每个人都要进行该检定。若至少有半数成员成功，则整个队伍成功。否则，整个队伍失败。

群体检定并不常见，但在整组角色要么一起成功、要么一起失败时很有用。例如，当冒险者正在穿越沼泽时，GM 可能会要求整队进行一次感知（生存）群体检定，以判断他们能否避开流沙、陷坑和其他自然危险。若至少有半数成员成功，那么成功者便能带领同伴脱离危险。否则，整个队伍就会跌入其中某种危险之中。

## 各项属性的使用

游戏中角色或怪物可能尝试的每一项任务，都归属于六项属性之一。本节会更详细地说明这些属性的含义，以及它们在游戏中的用法。

### 力量

力量衡量身体力量、运动训练，以及施加纯粹物理力量的能力。

#### 力量检定

力量检定可以模拟你举起、推动、拖拽或破坏某物的尝试，也可以模拟你让身体通过某个狭窄空间，或以蛮力应对某种局面的尝试。运动技能反映了某些特定力量检定中的天赋。

***运动***。力量（运动）检定涵盖你在攀爬、跳跃或游泳时遭遇的困难情境。示例如下：

- 你试图攀爬一面陡峭或湿滑的山崖，在翻越墙壁时避开危险，或在有东西试图将你击落时依旧抓住表面不放。
- 你试图跳出一段异常远的距离，或在跳跃途中做出高难动作。
- 你奋力在危险水流、惊涛骇浪或浓密海藻区域中游动或保持漂浮。或者，另一个生物试图把你推入水下、拉入水下，或以其他方式干扰你的游泳。

***其他力量检定***。GM 也可能在你尝试完成如下事情时要求进行力量检定：

- 强行打开一扇卡住、上锁或有门闩的门
- 挣脱束缚
- 挤过一条过于狭窄的隧道
- 在被拖在车后时抓住马车不放
- 推倒一座雕像
- 阻止一块巨石滚动

#### 攻击检定与伤害

当你以近战武器，例如硬头锤、战斧或标枪进行攻击时，要将力量调整值加入攻击检定和伤害掷骰中。你使用近战武器进行贴身战斗中的近战攻击，而其中一些武器也可投掷用于远程攻击。

#### 举重与负重

你的力量值决定了你能承受多少重量。以下术语定义了你能举起或携带的负重。

***负重上限***。你的负重上限等于力量值乘以 15。这个结果代表你能携带的重量（以磅计），通常足够高，所以多数角色通常不必太过担心。

***推、拖或举***。你最多可以推动、拖拽或举起等于你负重上限两倍的重量（也就是力量值的 30 倍，以磅计）。当你推动或拖拽超出自身负重上限的重量时，速度降为 5 尺。

***体型与力量***。更大的生物可以承受更多重量，而微型生物则能携带得更少。每高于中型一个体型类别，该生物的负重上限，以及它可推、拖或举的重量都会翻倍。对于微型生物，这些重量则减半。

#### 变体：负重

举重与负重规则故意设计得较为简单。若你想使用更细致的规则来决定角色会如何因装备重量受到妨碍，可使用以下变体。使用该变体时，忽略护甲表中的力量栏。

若你携带的重量超过力量值的 5 倍，则你处于 **负重** 状态，这意味着你的速度降低 10 尺。

若你携带的重量超过力量值的 10 倍，直至你的最大负重上限，则你改为处于 **重度负重** 状态，这意味着你的速度降低 20 尺，并且在使用力量、敏捷或体质进行的属性检定、攻击检定和豁免上具有劣势。

### 敏捷

敏捷衡量灵活性、反应速度和平衡能力。

#### 敏捷检定

敏捷检定可以模拟你灵巧、迅速、安静地移动的尝试，或避免在危险立足点跌倒的尝试。杂技、巧手和隐匿技能反映了某些特定敏捷检定中的天赋。

***杂技***。敏捷（杂技）检定涵盖你在棘手情况下试图保持站稳，例如试图跑过冰面、在绳索上保持平衡，或在摇晃船甲板上站稳。GM 也可能要求进行敏捷（杂技）检定，以判断你是否能完成杂技动作，包括俯冲、翻滚、空翻与转体。

***巧手***。每当你尝试进行手法把戏或手上花招时，例如把某样东西塞到别人身上，或将某件物品藏在自己身上，便进行一次敏捷（巧手）检定。GM 也可能要求进行敏捷（巧手）检定，以判断你是否能从别人身上顺走钱袋，或从别人口袋里摸出某样东西。

***隐匿***。当你试图将自己藏起来、潜过守卫、悄无声息地离开，或在不被看见或听见的情况下悄悄接近某人时，进行一次敏捷（隐匿）检定。

***其他敏捷检定***。GM 也可能在你尝试完成如下事情时要求进行敏捷检定：

- 在陡坡上控制一辆满载货物的车
- 驾驶战车绕过急弯
- 开锁
- 解除陷阱
- 牢固地捆住一个囚犯
- 扭动挣脱束缚
- 演奏弦乐器
- 制作一个小巧或精细的物件

#### 攻击检定与伤害

当你使用远程武器，如投石索或长弓进行攻击时，要将敏捷调整值加入攻击检定和伤害掷骰中。你也可以在使用具有灵巧属性的近战武器，例如匕首或刺剑攻击时，将敏捷调整值加入攻击检定与伤害掷骰中。

#### 护甲等级

根据你穿戴的护甲不同，你可能会将部分或全部敏捷调整值加入护甲等级。

#### 先攻

每场战斗开始时，你都要通过一次敏捷检定来掷先攻。先攻决定战斗中各生物的回合顺序。

> **隐藏**
>
>GM 会决定何种情势适合隐藏。当你尝试隐藏时，进行一次敏捷（隐匿）检定。在你被发现或停止隐藏之前，这次检定总值会与任何主动搜寻你踪迹的生物所进行的感知（察觉）检定进行对抗。
>
>你不能躲避一个能清楚看见你的生物；若你制造出声响，例如大声示警或打翻花瓶，也会暴露你的位置。
>
>隐形生物总是可以尝试隐藏。不过，它经过的痕迹仍可能被发现，而且它仍然必须保持安静。
>
>在战斗中，大多数生物都会保持警觉，注意四周危险迹象，因此若你从隐藏中现身并接近一个生物，它通常会发现你。不过，在某些情况下，若目标被分散了注意力，GM 可能允许你在靠近时继续保持隐藏，从而使你在被发现前的第一次攻击检定具有优势。
>
>***被动察觉***。当你隐藏时，即使别人没有主动搜寻，仍有可能察觉到你。为了判断某个生物是否察觉到你，GM 会将你的敏捷（隐匿）检定与该生物的被动感知（察觉）分数比较。该分数等于 10 + 该生物的感知调整值，以及任何其他加值或减值。若该生物具有优势，则加 5；若具有劣势，则减 5。例如，若一个 1 级角色（熟练加值为 +2）感知为 15（调整值为 +2），且熟练察觉，则其被动感知（察觉）分数为 14。
>
>***你能看见什么？*** 判断你能否找到一个隐藏的生物或物件，最主要的因素之一，是你在该区域中的视野状况，而该区域可能是 **轻度遮蔽** 或 **重度遮蔽**，详见“冒险”部分。

### 体质

体质衡量健康、耐力和生命力。

#### 体质检定

体质检定并不常见，而且没有任何技能适用于体质检定，因为这项能力所代表的耐力大多是被动体现，而非角色或怪物主动施为的某种技巧。不过，体质检定仍可用来模拟你试图超越自身极限的尝试。

GM 可能会在你尝试完成如下事情时要求进行体质检定：

- 屏住呼吸
- 长时间行军或劳动而不休息
- 熬夜不睡
- 在没有食物或饮水的情况下生存
- 一口气灌下一整大杯麦酒

#### 生命值

你的体质调整值会加入生命值。通常，你为生命值掷出的每一枚生命骰，都会加入体质调整值。

如果你的体质调整值发生变化，你的生命值上限也会随之变化，就好像你从 1 级开始便一直使用新的调整值一样。例如，若你在达到 4 级时提高了体质值，并使体质调整值从 `+1` 提升到 `+2`，则要按仿佛该调整值一直是 `+2` 的方式来调整你的生命值上限。因此，你前 3 个等级共增加 3 点生命值，而在 4 级时则用新的调整值来决定生命值。或者，若你是 7 级角色，而某种效果使你的体质值下降，从而令体质调整值降低 1，则你的生命值上限会降低 7。

### 智力

智力衡量精神敏锐度、记忆准确性与推理能力。

#### 智力检定

当你需要借助逻辑、教育、记忆或演绎推理时，便会用到智力检定。奥秘、历史、调查、自然和宗教技能反映了某些特定智力检定中的天赋。

***奥秘***。智力（奥秘）检定衡量你回忆有关法术、魔法物品、异界符号、魔法传统、存在位面及其居民知识的能力。

***历史***。智力（历史）检定衡量你回忆有关历史事件、传奇人物、古代王国、往昔纷争、近代战争和失落文明知识的能力。

***调查***。当你四处寻找线索，并基于这些线索进行推断时，要进行智力（调查）检定。你可能借此推断出隐藏物件的位置、从伤口外观判断造成它的是何种武器，或判断一条隧道最薄弱的位置从而导致其坍塌。在古卷中埋头翻找一段隐藏知识碎片，也可能需要进行智力（调查）检定。

***自然***。智力（自然）检定衡量你回忆有关地形、动植物、天气与自然循环知识的能力。

***宗教***。智力（宗教）检定衡量你回忆有关神祇、仪式与祈祷、宗教等级体系、圣徽以及秘密教团实践知识的能力。

***其他智力检定***。GM 也可能在你尝试完成如下事情时要求进行智力检定：

- 在不用语言的情况下与一个生物交流
- 估算一件贵重物品的价值
- 拼凑一身足以伪装成城卫兵的装束
- 伪造文书
- 回忆有关某门手艺或行业的知识
- 赢下一场技巧类游戏

#### 施法属性

法师使用智力作为施法属性，这有助于决定其法术的豁免 DC。

### 感知

感知反映你与周遭世界的协调程度，并体现察觉力与直觉。

#### 感知检定

感知检定可能体现你试图读懂肢体语言、理解他人情绪、注意环境中的细节，或照料伤者的努力。驯兽、洞悉、医药、察觉和生存技能反映了某些特定感知检定中的天赋。

***驯兽***。当你试图安抚一只驯养动物、避免坐骑受惊，或直觉判断某只动物意图时，GM 可能会要求进行感知（驯兽）检定。当你尝试让坐骑完成某个危险动作时，也要进行一次感知（驯兽）检定来控制坐骑。

***洞悉***。感知（洞悉）检定决定你能否判断一个生物的真实意图，例如识破谎言或预测某人的下一步举动。其过程通常包括从肢体语言、说话习惯和举止变化中提取线索。

***医药***。感知（医药）检定可以让你尝试稳定一位濒死同伴，或诊断某种疾病。

***察觉***。感知（察觉）检定能让你发现、听见，或以其他方式察觉某种存在。它衡量的是你对周围环境的总体警觉，以及感官的敏锐程度。例如，你可能尝试听见门后的交谈、偷听敞开窗下的谈话，或察觉森林中潜行怪物的动静。你也可能试图看见那些被遮蔽或容易忽略的事物，无论是埋伏在路边的兽人、躲在巷子阴影里的恶棍，还是一扇紧闭暗门下透出的烛光。

***生存***。GM 可能会要求你进行一次感知（生存）检定，以跟踪踪迹、猎取野味、引导队伍穿越冰封荒原、辨识附近有 owlbear 出没的迹象、预测天气，或避开流沙及其他自然危险。

***其他感知检定***。GM 也可能在你尝试完成如下事情时要求进行感知检定：

- 凭直觉判断该采取何种行动
- 分辨一个看似死去或活着的生物是否其实是不死生物

#### 施法属性

牧师、德鲁伊与游侠使用感知作为施法属性，这有助于决定其法术的豁免 DC。

### 魅力

魅力衡量你与他人有效互动的能力。它包括自信与口才等因素，也能体现一个人迷人或具有支配力的人格。

#### 魅力检定

当你试图影响或娱乐他人、试图留下印象或说出令人信服的谎言，或在复杂社交场合中周旋时，都可能会用到魅力检定。欺瞒、威吓、表演和游说技能反映了某些特定魅力检定中的天赋。

***欺瞒***。魅力（欺瞒）检定决定你是否能令人信服地掩盖真相，不论是通过言语还是行为。这种欺瞒可以包括利用模糊说辞误导他人，直到赤裸裸地撒谎。典型情境包括：对守卫油嘴滑舌、欺骗商人、通过赌博赚钱、借伪装冒充他人、用虚假的保证打消别人的怀疑，或在说一个明显谎言时保持面不改色。

***威吓***。当你试图通过公开威胁、敌对行动或身体暴力来影响某人时，GM 可能会要求进行魅力（威吓）检定。示例包括：从囚犯口中逼问情报、说服街头恶棍放弃冲突，或用碎酒瓶锋利边缘让一位冷笑着的大臣重新考虑决定。

***表演***。魅力（表演）检定决定你能否用音乐、舞蹈、表演、讲故事或其他娱乐形式取悦观众。

***游说***。当你试图用圆融手腕、社交礼仪或良善态度去影响一个人或一群人时，GM 可能会要求进行魅力（游说）检定。通常而言，你是在出于善意行事，以促进友谊、提出礼貌请求，或展现得体礼节时使用游说。游说他人的例子包括：说服总管让你的队伍觐见国王、在交战部族之间斡旋和平，或鼓舞一群城镇居民。

***其他魅力检定***。GM 也可能在你尝试完成如下事情时要求进行魅力检定：

- 找到最适合打听新闻、传闻与流言的人
- 混入人群以了解谈话中的关键话题

#### 施法属性

吟游诗人、圣武士、术士和邪术师使用魅力作为施法属性，这有助于决定其法术的豁免 DC。

# 豁免

豁免，也称为 save，代表你试图抵抗法术、陷阱、毒药、疾病或类似威胁的努力。你通常不会主动决定进行一次豁免；而是在角色或怪物面临伤害风险时，被迫进行豁免。

进行豁免时，掷一个 d20 并加入相应属性调整值。例如，进行敏捷豁免时，你使用敏捷调整值。

豁免可能会被情境加值或减值修正，也可能受到优势与劣势影响，具体由 GM 判定。

每个职业至少会给予两项豁免熟练。例如，法师熟练智力豁免。与技能熟练类似，豁免熟练允许角色将熟练加值加入以特定属性进行的豁免中。有些怪物也拥有豁免熟练。

豁免的难度等级由导致该豁免的效果决定。例如，由某个法术允许进行的豁免，其 DC 由施法者的施法属性与熟练加值决定。

豁免成功或失败的结果，也会在允许该豁免的效果中说明。通常，豁免成功意味着生物不受伤害，或只受较少伤害。
