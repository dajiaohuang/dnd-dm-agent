# 智能魔法物品

有些魔法物品拥有感知与人格。这样的物品可能被附身、被前任主人的灵魂萦绕，或因创造它的魔法而具有自我意识。无论是哪种情况，该物品都会像一个角色一样行动，拥有个性怪癖、理想、牵绊，有时还拥有缺陷。智能物品可能是其持用者珍贵的盟友，也可能是持续不断的麻烦。

大多数智能物品是武器。其他种类的物品也可以显现智能，但药水和卷轴等消耗品永远不会具有智能。

智能魔法物品作为 GM 控制下的 NPC 运作。该物品任何被激活的属性都由物品控制，而不是由其持用者控制。只要持用者与物品保持良好关系，持用者就能正常使用这些属性。若关系紧张，该物品可以压制其激活属性，甚至让这些属性反过来对付持用者。

## 创造智能魔法物品

当你决定让一件魔法物品具有智能时，你以创造 NPC 的方式创造该物品的人格，但这里描述了少数例外。

### 属性

智能魔法物品具有智力、感知和魅力属性值。你可以选择该物品的属性，也可以随机决定。若要随机决定，每项属性掷 4d6，弃掉最低骰并合计其余骰值。

### 沟通

智能物品具有某种沟通能力，可以传递情绪、以心灵感应广播思想，或出声说话。你可以选择它如何沟通，也可以在下表中掷骰。

**表：智能魔法物品：沟通**

| d100   | 沟通方式                                                                                   |
|--------|--------------------------------------------------------------------------------------------|
| 01-60  | 物品通过向携带或持用它的生物传递情绪来沟通。                                               |
| 61-90  | 物品能够说、读、理解一种或多种语言。                                                       |
| 91-100 | 物品能够说、读、理解一种或多种语言。此外，该物品能与携带或持用它的任何角色进行心灵感应沟通。 |
|        |                                                                                            |

### 感官

伴随智能而来的是意识。智能物品能在有限范围内感知周围环境。你可以选择它的感官，也可以在下表中掷骰。

**表：智能魔法物品：感官**

| d4 | 感官                         |
|----|------------------------------|
| 1  | 听觉，以及 30 尺范围的普通视觉。  |
| 2  | 听觉，以及 60 尺范围的普通视觉。  |
| 3  | 听觉，以及 120 尺范围的普通视觉。 |
| 4  | 听觉，以及 120 尺范围的黑暗视觉。 |
|    |                              |

### 特殊目的

你可以赋予智能物品一个它所追求的目标，甚至可能排斥除此以外的一切。只要持用者对该物品的使用符合这个特殊目的，该物品就会保持合作。偏离这个方向可能在持用者与物品之间引发冲突，甚至可能导致物品阻止其激活属性被使用。你可以选择一个特殊目的，也可以在下表中掷骰。

**表：智能魔法物品：特殊目的**

| d10 | 目的                                                                 |
|-----|----------------------------------------------------------------------|
| 1   | 阵营敌对：物品试图击败或毁灭与其阵营完全相反者。（此类物品绝不会是中立。） |
| 2   | 克星：物品试图击败或毁灭某一特定种类的生物，例如邪魔、变形怪、巨魔或法师。 |
| 3   | 守护者：物品试图保护某个特定种族或生物种类，例如精灵或德鲁伊。             |
| 4   | 十字军：物品试图击败、削弱或毁灭某位特定神祇的仆从。                     |
| 5   | 圣殿卫士：物品试图捍卫某位特定神祇的仆从与利益。                         |
| 6   | 毁灭者：物品渴求毁灭，并煽动其使用者任意战斗。                           |
| 7   | 追名逐誉者：物品试图成为世界上最著名的魔法物品，并让其使用者成为知名或恶名昭彰的人物。 |

### 冲突

智能物品具有由其个性与阵营塑造的自我意志。若其持用者以违背物品阵营或目的的方式行动，就可能发生冲突。当发生此类冲突时，该物品进行一次魅力检定，与持用者的魅力检定对抗。若物品赢得对抗，它会提出下列一项或多项要求：

- 物品坚持自己必须始终被携带或穿戴。
- 物品要求持用者丢弃任何它认为令人厌恶的东西。
- 物品要求持用者排除其他所有目标，只追求物品的目标。
- 物品要求被交给其他人。若持用者拒绝服从物品意愿，该物品可以执行下列任一项或全部行为：
- 使其持用者无法与其调谐。
- 压制一项或多项激活属性。
- 试图控制其持用者。

若智能物品试图控制其持用者，持用者必须进行一次魅力豁免，DC 等于 12 + 该物品的魅力调整值。若豁免失败，持用者被该物品魅惑 1d12 小时。被魅惑期间，持用者必须尝试遵循物品的命令。若持用者受到伤害，可以重复该豁免；若成功，效果结束。无论控制使用者的尝试成功还是失败，该物品直到下个黎明前都不能再次使用此能力。
