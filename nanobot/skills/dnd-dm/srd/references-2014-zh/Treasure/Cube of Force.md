### 力场方块

*奇物，稀有（需要调谐）*

这个方块边长约 1 寸。每个面上都有不同标记，可被按下。方块初始有 36 点充能，并在每日黎明恢复 1d20 点已消耗充能。

你可以用一个动作按下方块的某个面，并根据所选面消耗一定数量的充能，如“力场方块面”表所示。每个面都有不同效果。若方块剩余充能不足，则什么都不会发生。否则，一道隐形力场屏障会骤然出现，形成一个边长 15 尺的立方体。该屏障以你为中心，随你移动，并持续 1 分钟，直到你用一个动作按下方块的第六面，或方块耗尽充能。你可以按下方块的不同面并消耗所需充能来改变屏障效果，同时重置持续时间。

若你的移动使屏障接触到无法穿过方块的固体物件，只要屏障仍然存在，你就不能再靠近该物件。

**表：力场方块效果**

| 面 | 充能 | 效果 |
|----|------|------|
| 1  | 1    | 气体、风和雾不能穿过屏障。 |
| 2  | 2    | 非活物质不能穿过屏障。墙壁、地板和天花板可由你决定是否穿过。 |
| 3  | 3    | 活体物质不能穿过屏障。 |
| 4  | 4    | 法术效果不能穿过屏障。 |
| 5  | 5    | 任何东西都不能穿过屏障。墙壁、地板和天花板可由你决定是否穿过。 |
| 6  | 0    | 屏障失效。 |
|    |      |      |

当屏障成为某些法术的目标，或接触某些法术或魔法物品效果时，方块会失去充能，如下表所示。

**表：力场方块失去充能**

| 法术或物品 | 失去充能 |
|------------|----------|
| 解离术 | 1d12 |
| 爆破号角 | 1d10 |
| 穿墙术 | 1d6 |
| 虹光喷射 | 1d20 |
| 火墙术 | 1d4 |
|            |          |
