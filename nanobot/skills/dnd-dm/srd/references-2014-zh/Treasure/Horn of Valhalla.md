### 瓦尔哈拉号角

*奇物，稀有（银或黄铜）、非常稀有（青铜）或传奇（铁）*

你可以用一个动作吹响此号角。作为回应，来自瓦尔哈拉的战士灵魂会出现在你 60 尺内。它们使用狂战士的数据。它们会在 1 小时后或生命值降至 0 时返回瓦尔哈拉。你使用该号角后，直到经过 7 天前不能再次使用。

已知存在四种 *horn of Valhalla*，每种由不同金属制成。号角类型决定有多少狂战士响应其召唤，以及使用它所需的条件。GM 选择号角类型，或随机决定。

**表：瓦尔哈拉号角**

| d100  | 号角类型 | 召唤的狂战士 | 条件 |
|-------|----------|--------------|------|
| 01-40 | 银       | 2d4+2        | 无 |
| 41-75 | 黄铜     | 3d4+3        | 熟练所有简易武器 |
| 76-90 | 青铜     | 4d4+4        | 熟练所有中甲 |
| 91-00 | 铁       | 5d4+5        | 熟练所有军用武器 |
|       |          |              |      |

若你在不满足条件的情况下吹响号角，被召唤的狂战士会攻击你。若你满足条件，它们对你和你的同伴友好，并遵循你的命令。
