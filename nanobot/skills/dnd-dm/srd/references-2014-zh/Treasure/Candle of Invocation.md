### 祈神蜡烛

*奇物，非常稀有（需要调谐）*

这支细长蜡烛献给某位神祇，并具有该神祇的阵营。可以用 *detect evil and good* 法术侦测蜡烛的阵营。GM 选择该神祇和对应阵营，或随机决定阵营。

**表：祈神蜡烛**

| d20   | 阵营     |
|-------|----------|
| 1-2   | 混乱邪恶 |
| 3-4   | 混乱中立 |
| 5-7   | 混乱善良 |
| 8-9   | 中立邪恶 |
| 10-11 | 绝对中立 |
| 12-13 | 中立善良 |
| 14-15 | 守序邪恶 |
| 16-17 | 守序中立 |
| 18-20 | 守序善良 |
|       |          |

点燃蜡烛会激活其魔法，这需要一个动作。燃烧 4 小时后，蜡烛被毁灭。你可以提前将其熄灭，以便稍后使用。以 1 分钟为单位，从蜡烛总燃烧时间中扣除它已经燃烧的时间。

点燃期间，蜡烛在 30 尺半径内发出微光。该光照内任何阵营与蜡烛相同的生物，其攻击检定、豁免和属性检定具有优势。此外，该光照内阵营与蜡烛相同的牧师或德鲁伊，可以施放自己已准备的 1 环法术而不消耗法术位，但该法术的效果视为使用 1 环法术位施放。

或者，当你第一次点燃蜡烛时，你可以用它施放 *gate* 法术。如此做会毁灭该蜡烛。
