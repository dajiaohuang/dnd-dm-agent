### 祈祷珠项链

*奇物，稀有（需要由牧师、德鲁伊或圣武士调谐）*

这条项链有 1d4+2 颗由海蓝宝石、黑珍珠或黄玉制成的魔法珠。它还含有许多由琥珀、血石、黄水晶、珊瑚、玉、珍珠或石英等石料制成的非魔法珠。若一颗魔法珠从项链上取下，该珠便失去魔法。

魔法珠共有六种类型。GM 决定项链上每颗珠子的类型，或随机决定。一条项链可以有多颗同类型珠子。若要使用一颗珠子，你必须佩戴该项链。每颗珠子都含有一个法术，你可以用一个附赠动作从中施放（若需要豁免，则使用你的法术豁免 DC）。一颗魔法珠的法术施放后，该珠直到下个黎明前不能再次使用。

**表：祈祷珠项链**

| d20   | 珠子 | 法术 |
|-------|------|------|
| 1-6   | 祝福 | *bless* |
| 7-12  | 治疗 | *cure wounds*（2 环）或 *lesser restoration* |
| 13-16 | 恩宠 | *greater restoration* |
| 17-18 | 惩击 | *branding smite* |
| 19    | 召唤 | *planar ally* |
| 20    | 风行 | *wind walk* |
|       |      |      |
