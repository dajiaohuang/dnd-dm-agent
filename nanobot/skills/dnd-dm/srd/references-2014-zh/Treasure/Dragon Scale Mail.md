### 龙鳞甲

*护甲（鳞甲），非常稀有（需要调谐）*

龙鳞甲由某一种龙的鳞片制成。有时，龙会收集自己脱落的鳞片并将其赠予类人生物。另一些时候，猎人会仔细剥下并保存死龙的皮。无论哪种情况，龙鳞甲都极受珍视。

当你穿着此护甲时，你的 AC 获得 +1 加值，你对抗龙的可怖威仪和吐息武器的豁免具有优势，并且你对一种伤害类型具有抗性，该类型由提供鳞片的龙种决定（见下表）。

此外，你可以用一个动作集中感官，以魔法方式辨别你 30 英里内最近的、与该护甲同类型龙的距离和方向。此特殊动作直到下个黎明前不能再次使用。

**表：龙鳞甲**

| 龙 | 抗性 |
|:---|:-----|
| 黑龙 | 强酸 |
| 蓝龙 | 闪电 |
| 黄铜龙 | 火焰 |
| 青铜龙 | 闪电 |
| 赤铜龙 | 强酸 |
| 金龙 | 火焰 |
| 绿龙 | 毒素 |
| 红龙 | 火焰 |
| 银龙 | 寒冷 |
| 白龙 | 寒冷 |
