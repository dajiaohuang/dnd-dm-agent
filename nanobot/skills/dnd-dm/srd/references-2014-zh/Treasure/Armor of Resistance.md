### 抗性护甲

*护甲（轻甲、中甲或重甲），稀有（需要调谐）*

当你穿着此护甲时，你对一种伤害类型具有抗性。GM 选择该类型，或从下列选项中随机决定。

**表：抗性护甲**

| d10 | 伤害类型 |
|-----|----------|
| 1   | 强酸     |
| 2   | 寒冷     |
| 3   | 火焰     |
| 4   | 力场     |
| 5   | 闪电     |
| 6   | 黯蚀     |
| 7   | 毒素     |
| 8   | 心灵     |
| 9   | 光耀     |
| 10  | 雷鸣     |
|     |          |
