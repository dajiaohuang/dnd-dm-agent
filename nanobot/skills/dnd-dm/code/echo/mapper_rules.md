# 回声映射规则 (Echo Mapper Rules)

## 核心原则

```
玩家世界的每一件事 = 阿弗纳斯的一枚灵魂币
流通方式不同，价值不变
```

### 人物映射规则

现实中的每个人物 → D&D NPC 时按以下维度调整：

```
角色职能   →  类职业对应
  上司     →  暴君/巫妖/大商人
  同事     →  竞争者/盟友/间谍
  家人     →  家族长者/守护灵
  朋友     →  冒险同伴/酒馆老板
  陌生人   →  路边旅人/神秘商人
  对手     →  邪教徒/赏金猎人

关系强度  →  NPC 关联度
  亲密     →  血脉/契约绑定
  普通     →  偶遇/委托关系
  疏远     →  传闻/背景提及

冲突深度  →  任务层级
  表面     →  支线任务
  深层     →  主线分支
  核心     →  章节级故事
```

### 战利品映射

现实满足感来源 → 游戏内奖励对应：

```
完成工作   →  金币/经验
获得认可   →  声望/头衔
学习成长   →  新法术/专长
人际关系   →  盟友/追随者
物质获得   →  魔法物品
自由时间   →  安全营地/传送阵
```

## 映射质量规则

1. **隐喻原则**：不直接照搬，必须转换。老板→巫妖可以，老板→老板不行
2. **情感保真**：保留原始情绪强度。玩家焦虑程度决定任务紧迫度
3. **匿名保护**：NPC 名字不映射、具体事件细节模糊化
4. **喜剧缓冲**：处理敏感话题时可添加幽默元素缓冲（但不可冲淡核心隐喻）
5. **解密可选**：结束后是否揭示隐喻映射归玩家决定
